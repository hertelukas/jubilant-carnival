# Jubilant C# Server

## Options
| Name               | Default | Description                                           |
|--------------------|---------|-------------------------------------------------------|
| `--help`           |         | Get help                                              |
| `--listip`         |         | Get a list of available IP addresses to host on       |
| `--ip=<Integer>`   | `0`     | Specify the IP address to listen on. (See `--listip`) |
| `--port=<Integer>` | `36187` | Specify the port                                      |
